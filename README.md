# Rahu Kalaya Calculator JS 🕒

A lightweight, zero-dependency JavaScript library to calculate daily Rahu Kalaya (Rahu Kaal) for Sri Lanka (and worldwide based on sunrise/sunset variations).

## Live Demo
Check out the live working tool here: [Rahu Kalaya Sri Lanka Today](https://tools.unicodesinhala.com/rahu-kalaya)


## CDN Usage ⚡

You can easily integrate **Rahu.js** into your website using jsDelivr CDN. Just add the following script tag to your HTML file:

```html
<!-- Load Rahu.js from jsDelivr CDN -->
<script src="[https://cdn.jsdelivr.net/gh/xzunk/rahu@main/rahu.min.js](https://cdn.jsdelivr.net/gh/xzunk/rahu@main/rahu.min.js)"></script>
